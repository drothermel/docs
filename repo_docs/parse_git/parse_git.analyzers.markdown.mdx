## <kbd>class</kbd> `MarkdownAnalyzer`
Analyzes the lifecycle and evolution of markdown files in a Git repository.

This class tracks markdown file creation, modification, deletion, and renaming
across a specified time period, building comprehensive lifecycle data including
milestones and associated code commits.
