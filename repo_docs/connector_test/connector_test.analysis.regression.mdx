```python
regression_analysis(
    mean_curve: 'ndarray',
    log_scale: 'bool' = True,
    window: 'int' = 5,
    intervals: 'dict[str, tuple[int, int]] | None' = None
) → tuple[dict[str, float], ndarray, ndarray, list[tuple[int, int, float, float]]]
```

Compute regression fits for different epoch ranges.

Parameters
----------
mean_curve:
    Sequence of metric values to fit.
log_scale:
    Whether to perform the regression in log space.
window:
    Sliding window size for non-parametric fits.
intervals:
    Mapping of interval names to ``(start, end)`` epoch pairs used for
    reporting separate linear fits.
