## <kbd>function</kbd> `get_cifar_dataset`

```python
get_cifar_dataset(
    dataset_name: str,
    source_split: str,
    root: str,
    transform: Any | None = None,
    download: bool = True
) → Dataset[Any]
```

Loads a specified dataset (CIFAR-10 or CIFAR-100).


**Args:**

- <b>`dataset_name`</b> (str): Name of the dataset ("cifar10" or "cifar100").
- <b>`source_split`</b> (str): "train" or "test"
- <b>`root`</b> (str): Root directory where the dataset is stored or will be downloaded.
- <b>`transform`</b> (callable, optional): A function/transform to apply to the data.
- <b>`download`</b> (bool): Whether to download the dataset if not found locally.


**Returns:**

- <b>`torch.utils.data.Dataset`</b>: The loaded dataset.

## <kbd>class</kbd> `TransformedSubset`
A wrapper for a torch.utils.data.Subset that applies a transform.

This is useful because Subsets themselves don't have a transform attribute
that can be set after creation.
