## Generating These Docs

Use the local fork of [lazydocs](https://github.com/ml-tooling/lazydocs) which supports python 3.12 and makes source code links and formatting work with mintlify.

For each repo you want to be documented:
```bash
MYREPO="new_shiny_repo"
cd ${MYREPO}
uv add ../lazydocs --editable

cd ../docs/
source ../${MYREPO}/.venv/bin/activate

lazydocs --output-path repo_docs/${MYREPO}/ --overview-file ${MYREPO} \
         --src-base-url https://github.com/drothermel/${MYREPO}/blob/main/src/ \
         --output-format mdx --toc --private-modules ${MYREPO}
```

Then to see locally:
```
mint dev
```

And the mintlify github bot will handle deploying to [my mintlify](https://dr-a2c58821.mintlify.app/) (or see [dashboard](https://dashboard.mintlify.com/dr-a2c58821/dr-a2c58821)).
